CodeIgniter 3 Admin
-------------------------
一个基于 [Codeignter](https://github.com/bcit-ci/CodeIgniter) 和 [AdminLTE](https://github.com/almasaeed2010/AdminLTE) 的管理后台项目。   

**CI Version:** ```3.1.10```   

-------------------------
## 使用方法
- 项目入口为 ``public``
- 导入 ``db/db.sql`` 管理员账号：```admin```，密码：```123456```
- 关于 ```nginx``` 配置，可参考 ```nginx/*.conf``` 两个文件
